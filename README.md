# 🍽️ Restaurant Website

A modern responsive restaurant website built with React + Vite.

## 🚀 Features
- Responsive landing page
- Multi-page routing (Home, Menu, About, Contact)
- Reusable components
- Theme toggle support
- Scroll animations
- Clean UI/UX design

## 🛠 Tech Stack
- React
- Vite
- CSS
- JavaScript
